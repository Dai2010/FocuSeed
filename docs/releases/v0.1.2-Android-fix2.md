# FocuSeed v0.1.2-Android-fix2

本次只发布 Android APK，重点修复部分设备打开应用后立即闪退、系统提示“屡次停止”并杀掉应用的问题。

## 修复内容

- 启动时不再主动拉起旧会话的守护服务，避免前台服务/通知权限异常触发系统连续杀进程。
- 守护服务增加安全降级：前台通知、闹钟、电话监听、通知取消异常时不再让主进程崩溃。
- 旧版本残留的异常番茄钟设置会自动清洗为默认值，避免恢复会话时崩溃。
- 字体加载、拨号器检测、勿扰权限读取、全屏通知权限读取失败时使用降级文案继续启动。
- 首次权限提示和更新提示增加 Activity 状态保护，避免窗口状态异常导致闪退。

## 安装说明

- APK `versionCode` 升级到 5，继续使用同一发布签名，可从 v0.1.1/v0.1.2/v0.1.2-Android-fix1 直接覆盖安装。
- 如果系统仍提示旧版本屡次停止，先在系统设置里“强行停止” FocuSeed，再安装本版本覆盖。

## 已知限制

- 普通 Android 应用无法在所有品牌系统上完全屏蔽系统手势。本版本先保证不闪退，再通过全屏、屏幕固定、通知和闹钟尽力拉回专注状态。
