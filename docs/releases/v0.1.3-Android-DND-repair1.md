# FocuSeed v0.1.3 Android DND Repair 1

这是单独提供的 Android 勿扰修复包，不是主应用更新包。

## 适用问题
- 部分用户在测试早期 Android 版本后，系统免打扰/勿扰模式下媒体音量失效。
- 原因是早期版本曾修改系统勿扰策略为“电话优先”，某些国产 ROM 可能会长期保留该策略。

## 使用方法
- 安装 `FocuSeed-DND-Repair-v0.1.3-Android-DND-repair1-android.apk`。
- 打开“FocuSeed 勿扰修复包”。
- 点击“一键修复媒体音量”。
- 如果系统要求授权，请允许“FocuSeed 勿扰修复包”访问勿扰设置，返回后会自动继续修复。
- 修复完成后可以卸载本修复包；如果仍需要系统勿扰模式，请在确认媒体音量恢复后手动重新开启。

## 修复内容
- 关闭一次由 FocuSeed 早期版本可能留下的勿扰状态。
- 将媒体、闹钟、电话加入系统勿扰允许项。
- 独立包名：`io.github.dai2010.focuseed.repair`，不会覆盖主应用。
- 仅发布 Android APK，不发布桌面端包。
