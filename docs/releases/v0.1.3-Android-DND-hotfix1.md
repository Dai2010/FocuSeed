# FocuSeed v0.1.3-Android-DND-hotfix1

这是 Android 单独紧急修复包，明确用于修复旧测试版可能改坏系统免打扰策略，导致免打扰模式下媒体音量失效的问题。

## 必须尽快升级

- 旧版曾调用系统勿扰策略，只放行电话，未放行媒体/闹钟。
- 在部分系统，尤其是华为/国产 ROM 上，这个策略会持续影响系统免打扰模式下的媒体声音。
- 本版在主界面提供醒目的“一键修复媒体音量”按钮，安装后打开应用点击即可修复。

## 一键修复做了什么

- 重新写入 FocuSeed 的勿扰策略：放行电话、媒体和闹钟。
- 不再自动切换系统到免打扰模式，避免继续影响用户现有系统声音设置。
- 如果尚未授权“勿扰访问”，按钮会引导到系统设置；授权后返回 FocuSeed 再点一次即可。

## 诊断信息

- 这是 Canary 诊断修复包，内置应用内崩溃记录和“复制 Canary 诊断报告”按钮。
- 华为手机无需 root、无需电脑；如仍异常，复制诊断报告发给开发者即可。

## 安装说明

- APK `versionCode` 升级到 6，继续使用同一发布签名，可直接覆盖安装。
- 发布资产只包含 Android APK：`FocuSeed-v0.1.3-Android-DND-hotfix1-android.apk`。
