# FocuSeed v0.1.2-Android-fix1

这是只面向 Android 的修复发布，桌面端安装包不随本标签重新发布。

## 修复重点

- 修复工作期使用手势上滑即可离开应用后无法稳定回到专注界面的问题。
- 将后台拉回逻辑迁移到前台守护服务，避免 Activity 反复自启动被系统判定为异常退出。
- 应用内临时退出和系统手势退出都会消耗同一组 3 次机会，计时继续按原始开始时间运行。
- 休息期仍允许退出；下一段工作开始时由守护服务、闹钟和全屏通知共同提示回到专注。
- 拨号、权限设置、更新安装路径保留临时放行，通话结束或返回后继续恢复番茄钟状态。
- APK `versionCode` 升级到 4，继续使用同一发布签名，可从 v0.1.1/v0.1.2 直接覆盖安装。

## 已知限制

- 普通 Android 应用无法在所有厂商系统上完全禁止系统手势；本版改为使用前台服务、屏幕固定、AlarmClock 和全屏通知组合增强拉回可靠性。
- 如系统弹出“屏幕固定”或通知/全屏通知权限提示，请允许，否则强制全屏能力会受系统策略限制。
