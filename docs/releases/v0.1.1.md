# FocuSeed v0.1.1

FocuSeed v0.1.1 是一次界面、专注体验和更新机制补强：界面更柔和可爱，支持休息退出/工作自动回全屏，并加入启动时静默检查更新与下载进度提示。

## 新增内容

- 新增中文萌系字体 ZCOOL KuaiLe，Android 和桌面端都会随应用内置字体文件。
- Android 界面改为柔和渐变背景、圆角卡片、低刺激色彩和更可爱的文案。
- Windows/Linux 桌面界面改为柔和渐变背景、圆角卡片和可爱中文字体。
- 使用项目图标作为 Android 启动器图标、Windows `.exe/.msi` 图标、Linux DEB/RPM 图标和 Arch Linux 桌面图标。
- 休息期间允许用户退出或隐藏应用，下一段工作时间开始后自动恢复全屏。
- 工作期间提供 3 次短暂退出机会，计时不会暂停；机会用完后继续保持全屏。
- Android 端保存会话时间锚点，应用被移到后台后仍按真实时间恢复番茄钟状态。
- Android、Windows、Linux 启动时静默检查 GitHub Release；发现新版本后提示用户下载对应平台安装包。
- 默认使用 GitHub 原始 release asset 地址下载到系统下载目录；如果下载卡住，用户可以手动切换到 `https://ghfast.top/` 加速地址并查看进度。
- Android APK 改为固定 release keystore 签名，后续同包名 APK 可以直接覆盖更新。

## 已知限制

- Android 从后台自动回到全屏依赖系统闹钟/Activity 启动策略，不同厂商 ROM 可能会有额外限制。
- 已安装早期 debug APK 的用户可能需要先卸载旧包；从本次覆盖后的签名 APK 起，后续版本会保持同一签名。
- 桌面端退出/隐藏逻辑属于应用级约束；Windows/Linux 严格 Kiosk 仍需要系统策略配合。
- Linux GNOME/Wayland 等环境仍只能提供最佳努力全屏。
