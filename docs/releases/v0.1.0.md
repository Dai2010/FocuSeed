# FocuSeed v0.1.0

FocuSeed v0.1.0 是项目的第一个公开版本，重点提供番茄钟核心计时、全屏专注界面、Android 拨号入口和跨平台安装包发布链路。

## 新增内容

- Android 10+ 预览版 APK：支持沉浸式全屏、勿扰权限引导、电话例外策略、系统拨号器检测和拨号跳转。
- Windows 桌面预览版：提供全屏番茄钟窗口，并发布 `.exe` 与 `.msi` 安装包。
- Linux 桌面预览版：提供全屏番茄钟窗口，并发布 DPKG/DEB、RPM 安装包。
- Arch Linux 版本：发布 `.pkg.tar.zst` 包。
- 计时模型：基于真实时间锚点，应用失焦、进入后台或跳转拨号器期间不暂停计时。
- 用户设置：支持自定义工作时长、休息时长和轮数。

## 已知限制

- Android 的严格锁定能力依赖用户授权或设备所有者/Kiosk 配置，普通应用不能无条件阻止 Home、最近任务或系统切换。
- Windows 真正阻止应用切换需要 Assigned Access 或管理员策略配合。
- Linux GNOME/Wayland 等环境只能提供最佳努力全屏，强制占屏需要 Kiosk 会话。
- Android APK 当前使用调试签名，适合测试安装，不作为应用商店正式签名包。
