# FocuSeed v0.1.2

FocuSeed v0.1.2 重点修复 Android 手势上滑可直接离开的问题，并同步精简 Android、Windows 和 Linux 的主界面。

## 新增内容

- Android 工作期会尝试进入系统屏幕固定/Lock Task，降低手势上滑、任务切换等方式绕过全屏专注的风险。
- Android 工作期被意外切到后台时会去重处理生命周期事件，避免一次手势重复扣除退出机会，并按规则快速拉回全屏。
- Android 首次运行会集中检测电话状态、通知和勿扰相关权限；后续启动不再自动反复询问，用户可在左上角菜单手动设置。
- Android、Windows 和 Linux 的设置项、权限状态、平台信息和更新入口收纳进左上角折叠菜单，主界面更简洁。
- 桌面端新增最小化防护：工作期最小化会消耗临时退出机会，用完后立刻恢复全屏；休息期仍允许隐藏到下一段工作。
- 界面文案改用纯文本颜文字，不再使用 emoji，减少旧系统或缺字字体显示异常。

## 修复内容

- 修复 Android 上通过手势上滑可能直接离开应用且不立即恢复的问题。
- 修复 Android 生命周期多次触发时可能重复扣除临时退出机会的问题。
- 修复桌面端通过最小化绕过全屏专注状态的问题。

## 已知限制

- Android 普通应用无法像设备所有者/Kiosk 一样绝对禁止所有系统级手势；本版会优先使用系统屏幕固定并配合闹钟/Activity 拉回。
- 如果用户未开启或拒绝系统屏幕固定确认，不同厂商 ROM 仍可能只支持最佳努力拉回。
- Windows/Linux 桌面端仍属于应用级约束；严格锁定需要系统 Kiosk、Assigned Access 或桌面策略配合。
- Linux GNOME/Wayland 等环境仍可能限制普通应用独占全屏或置顶。
