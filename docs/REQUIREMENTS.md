# FocuSeed 需求说明

## 产品定位

FocuSeed 是一个强制专注型番茄钟。它通过全屏、通知抑制、电话例外和跨平台计时恢复，帮助用户在工作周期内减少主动分心。

## 用户可配置项

- 工作时长：每轮番茄钟的专注时间。
- 休息时长：每轮之间的短休息时间。
- 轮数：一次专注计划中包含多少轮番茄钟。
- 严格程度：普通全屏、尽力锁定、系统 Kiosk/设备所有者模式。

## 番茄钟状态

- Idle：未开始。
- Working：工作中。
- ShortBreak：休息中。
- Finished：全部轮次完成。
- InterruptedByCall：Android 通话中或拨号器前台中。

## 计时规则

- 计时基于会话开始时间、阶段开始时间和当前真实时间计算。
- 应用进入后台、打开拨号器、接听电话或桌面窗口失焦时，计时不暂停。
- 恢复到前台时根据真实时间重新计算当前阶段、剩余时间和轮次。
- 如果用户在离开期间跨过多个阶段，应用应直接跳转到当前应处于的阶段。

## Android 电话例外

- 番茄钟运行中应允许接听来电。
- 应用内提供拨号入口，让用户主动打开系统拨号器。
- 拨号器检测不能依赖固定厂商包名，需要基于系统 API 自动发现。
- 用户从拨号器返回或通话结束后，应恢复番茄钟全屏界面。

## 桌面端要求

- Windows 和 Linux 都应提供全屏专注界面。
- 桌面端应尽量置顶和隐藏窗口装饰。
- 桌面端无法获得电话例外能力，电话相关入口仅在 Android 显示。
- Linux 必须清晰提示当前桌面环境是否只能提供最佳努力全屏。
