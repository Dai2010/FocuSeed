# FocuSeed Icons

These icons are generated from the project icon provided by the project owner as `FocuSeed.png`.

- `focuseed.png`: 1024×1024 PNG source used for generated packages.
- `focuseed-256.png`: Linux/desktop package icon.
- `focuseed.ico`: Windows installer and executable icon.

Android launcher icons are generated into `androidApp/src/main/res/mipmap-*`.
